A modern and responsive game discovery web application built with **Angular**. Browse thousands of games, filter them by category and platform, and explore detailed information using the FreeToGame API.

## ✨ Features

- 🎮 Browse free-to-play games
- 🔍 Filter games by category
- 💻 Browse games by platform
- 📄 View detailed game information
- ⚡ Fast API integration
- 📱 Fully responsive design
- 🔄 Dynamic routing with Angular Router
- ⏳ Loading indicators for better user experience

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AlamirSamir/GameOver.git
```

Go to the project folder

```bash
cd GameOver
```

Install dependencies

```bash
npm install
```

Run the application

```bash
ng serve
```

Open your browser and visit

```
http://localhost:4200
```
